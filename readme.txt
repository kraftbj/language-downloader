== Description ==

Allows for on-the-fly downloading of new translations using WordPress 4.0 Translation API improvements.

== Installation ==

1. Upload contents of the directory to /wp-content/plugins/ (or use the automatic installer)
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Visit Tools->Language Downloader and select a language to download.
1. Once downloaded, you can switch to the language via Settings->General

== Frequently Asked Questions ==

= This plugin is lacking! =

Yup. It's a quick plugin to add this functionality to WordPress. It is in active development and improvements are coming. I fully expect Core to offer something better than this directly in 4.1.
